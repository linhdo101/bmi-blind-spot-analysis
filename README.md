# The BMI Blind Spot: Rethinking BMI as a Health Metric

## 📌 Overview
This project critically evaluates Body Mass Index (BMI) as a health indicator and investigates its limitations in capturing true body composition and metabolic health.

## 🎯 Objective
To assess whether BMI is a reliable metric for classifying obesity and health risks by comparing it with alternative indicators such as fat ratio, muscle mass, and metabolic variables.

## 🧠 Approach
- Data cleaning and preprocessing (318 patient records)
- Exploratory Data Analysis (BMI distribution, composition metrics)
- Statistical testing:
  - Mann-Whitney U Test
  - Kruskal-Wallis Test
  - Chi-square Test
- Regression analysis to evaluate BMI as an explanatory variable

## 📊 Key Findings
- BMI is weak in explaining metabolic health indicators
- Individuals with similar BMI can have very different body compositions
- BMI fails to capture internal health risks (e.g., comorbidity)
- Strong association observed only with liver fat accumulation

## 🚨 Key Insight
BMI oversimplifies health and can lead to misleading classifications and social bias.
